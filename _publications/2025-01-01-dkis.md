We propose **DKiS**, an invertible image steganography framework that addresses two key limitations of existing methods:

1. **Decay Weight Mechanism** — regulates the transfer of non-essential "garbage" information from the secret to the host pipeline, improving steganographic quality and robustness.
2. **Private Key Integration** — incorporates a preset private key into high-capacity image steganography for the first time. Access to concealed data requires the corresponding private key, significantly strengthening security against adversarial extraction.

DKiS provides a practical solution for secure, high-capacity image steganography applicable in real-world information hiding scenarios.

Code available at [https://github.com/yanghangAI/DKiS](https://github.com/yanghangAI/DKiS).
